האתר של אורי — קריאה בתורה (פרשת ויצא, בראשית ל׳ י״ד–כ״ז)
======================================================================

איך פותחים
----------
פשוט לוחצים פעמיים על הקובץ  index.html  והוא נפתח בדפדפן (Chrome/Edge).
אפשר גם להעלות את כל תיקיית ה-site לאירוח אינטרנטי חינמי (Netlify / GitHub Pages)
כדי לקבל קישור שאורי יוכל לפתוח מכל מכשיר.

מה יש באתר
----------
• כל 14 הפסוקים, שורה לכל פסוק, עם ניקוד וטעמים.
• ליד כל פסוק כפתור ▶ — לוחצים ושומעים רק את הפסוק הזה מתוך ההקלטה המלאה,
  והמילה שנקראת מודגשת בצהוב בזמן אמת.
• "נגן הכול מההתחלה" — משמיע את כל הקטע ברצף עם ההדגשה.
• בקרת מהירות (0.7× / 0.85× / 1×) — נוח להאטה בזמן לימוד.
• כפתור נפרד לשמיעת ההקלטה הקצרה של שני הפסוקים הראשונים.
• פתיחת תמונת הקטע המקורי.

על דיוק ההדגשה
--------------
זמני הפסוקים והמילים חושבו אוטומטית מתוך ההקלטה (זיהוי דיבור + יישור לטקסט).
הגבולות בין פסוקים מדויקים מאוד. ההדגשה בתוך הפסוק ברמת המילה מדויקת ברובה,
אך במילים ארוכות עם מנגינה מתמשכת ייתכן סטייה קטנה.

איך מתקנים תזמון (אם צריך) — "מצב סנכרון (להורה)"
--------------------------------------------------
1. לוחצים על "⚙️ מצב סנכרון (להורה)".
2. בוחרים פסוק מהרשימה.
3. לוחצים "🎯 הקלט תזמון מילים" — ההקלטה מתחילה להתנגן.
4. מקישים על מקש הרווח (או על הכפתור הגדול) בדיוק ברגע שכל מילה מתחילה להישמע.
5. בסוף המילה האחרונה מקישים רווח פעם נוספת כדי לסמן את סיום הפסוק.
   → התזמון החדש נשמר אוטומטית בדפדפן.
• אפשר גם להקליד ידנית זמן התחלה/סיום לפסוק וללחוץ "עדכן גבולות".
• "⬇️ ייצוא כל הזמנים (JSON)" יוריד קובץ timings.js — אם מחליפים בו את הקובץ
  הקיים בתיקייה, התיקונים נשמרים לתמיד וגם עבור כל מי שיפתח את האתר.

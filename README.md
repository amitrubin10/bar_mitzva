# הקריאה של אורי בתורה 🎉

אתר לימוד קריאה בתורה בטעמים לכבוד בר המצווה — **פרשת ויצא, עלייה רביעית** (בראשית ל׳ י״ד–כ״ז).

## מה יש כאן
- כל 14 הפסוקים, שורה לכל פסוק, עם ניקוד וטעמים.
- כפתור ▶ ליד כל פסוק — משמיע רק את אותו פסוק מתוך ההקלטה, **והמילה הנקראת מודגשת בזמן אמת**.
- "נגן הכול" עם הדגשה רציפה, ובקרת מהירות (0.7× / 0.85× / 1×) ללימוד איטי.
- הקלטה נפרדת של שני הפסוקים הראשונים, ותמונת הקטע המקורי.
- **PWA** — ניתן להתקנה כאפליקציה בטלפון ועובד גם ללא אינטרנט (offline).

## שימוש
פשוט פותחים את הכתובת בדפדפן. בטלפון אפשר "הוסף למסך הבית" כדי להתקין כאפליקציה.

## כוונון תזמון (להורה)
זמני הפסוקים והמילים חושבו אוטומטית מההקלטה. לדיוק מושלם ברמת המילה יש
**"מצב סנכרון (להורה)"** — בוחרים פסוק, מנגנים, ומקישים רווח בתחילת כל מילה.
